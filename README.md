# CREDITFLOW

CreditFlow è un gestionale per la gestione dei clienti e delle relative pratiche bancarie, con funzionalità di ricerca, filtro, ordinamento e visualizzazione dettagliata di ogni pratica.

# Funzionalità principali
- Visualizzazione delle pratiche in una tabella ordinabile e filtrabile
- Ricerca per cliente e banca
- Filtri per stato delle pratiche (In lavorazione, Erogata, Fatturata, Deliberata)
- Ordinamento per cliente, banca, importo, tipologia, stato e data acquisizione
- Pannello dettaglio pratica con tutte le informazioni
- Creazione, modifica ed eliminazione delle pratiche in db
- Notifiche visive per operazioni riuscite

# Tecnologie utilizzate

- Frontend: React, Bootstrap, React-Select
- Backend: Node.js, Express, MySQL
- Gestione ambienti: dotenv
- Versionamento: Git & GitHub
